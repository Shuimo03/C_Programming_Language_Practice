### 赋值和初始化

- 和数学不同，a = b在数学中表示关系，而在程序设计中， a = b表示动作，关系是静态的，动作是动态的，在数学中，a = b和b = a完全相同，但是在程序设计中就不可以，两者意思完全相反。
- 当赋值发生在定义变量的时候，就是变量的初始化，虽然C语言并没有强制要求所有的变量都在定义的地方做初始化，但是所有的变量在第一次被使用(出现在赋值运算符右边)之前就应该被赋值一次。

### while循环

- 把while循环翻译成当，那么一个while循环的意思就是：当条件满足的时候，不断重复循环体内的语句。
- 循环执行之前判断是否继续循环，所以有可能循环一次也没有被执行；
- 条件成立是循环继续的条件。

### do-while循环

在进入循环的时候不做检查，而是在执行完一轮循环体的代码之后，再来检查循环的条件是否满足，如果满足则继续下一轮循环，不满足则结束循环，至少执行一次。

do-while和while的区别在于，do-while的循环体执行结束的时候才判断条件，与while相同的时候在于，条件满足的时候执行循环，不满足的时候结束循环。

### for循环

for循环像一个计数循环，设定一个计数器，初始化它，然后计数器到达某值之前，重复执行循环体，而每执行一轮循环，计数器值以一定步进进行调整，比如加1或减1。

### 循环的选择

for循环可以和while互相改写，改写如下

```
for(初始动作;条件;每轮的动作){} = for(;条件;) == while(条件)
```

- 如果有固定次数，用for循环
- 如果必须执行一次，用do_while循环
- 其他情况用while循环

### 循环控制

- break:跳出循环
- continue:跳过循环这一轮剩下的语句进入下一轮

### 短路机制

- 逻辑运算是自左向右进行，如果左边的结果可以决定结果，就不会执行右边计算。
- 对于&&来说，左边是false的时候就不做右边了。
- 对于||来说，左边是true的时候就不做右边了。

### 条件运算符

条件?条件满足的值:条件不满足的值，使用如下，条件运算符是自右向左结合的。

```c
#include<stdio.h>

int main(){

    int a = 10;
    int b = 20;
    int res = (a > b) ? 1:0;
    //相当下面这个if语句
    if(a > b){
        res = 1;
    }
    else{
        res = 0;
    }
    printf("%d\n", res);
    return 0;
}
```

条件运算符的优先级高于赋值运算符，但是低于其他运算符。

### 逗号运算符

逗号用来连接两个表达式，并以其右边的表达式的值作为它的结果。逗号的优先级是所有的运算符中最低的，所以它两边的表达式会先计算；逗号的组合关系是自左向右，所以左边的表达式会先计算，而右边的表达式的值就留下作为逗号运算的结果，主要用在for循环。

### 多路分支 switch-case

```c
switch(控制表达式){ 控制表达式只能是整数结果
	case 常量: 常量可以是常数，也可以是常数表达式
	语句
	....
	case 常量:
	语句
	....
    default:
    语句
    ....
}

#include<stdio.h>

int main(){

    int type = 0;
    scanf("%d", &type);

    switch(type){

        case 1:
            printf("你好");
            break;
        case 2:
            printf("早上好");
            break;
        case 3:
            printf("下午好");
            break;
        case 4:
            printf("再见");
            break;
        default:
            printf("啊?什么啊?");
    }
    return 0;
}
```

switch语句可以看作是一种基于计算的跳转，计算控制表达式的值后，程序会跳到相匹配的case处，分支标号只是说明了switch内部位置的路标，在执行完分支中最后一条语句后，如果没有后面的break，就会顺序执行到下面的case里面去，知道遇到下一个break或者switch结束为止。

### 函数

函数是一块代码，接受零个或多个参数，做一件事情，并且返回零个或一个值。

```c
int count(int n){ //函数头 = 返回类型 函数名 参数表
    int res = 0;
    for (int i = 1; i <= n; i++){ //{}内是函数体
        res += i;
    }
    return res;
}
```

调用函数是函数名(参数值)

- ()起到了表示函数调用重要性
- 即使没有参数也需要()
- 如果有参数，则需要给出正确的数量和顺序
- 这些值会按照顺序依次用来初始化函数中的参数
- 函数知道每一次是哪里调用它，会回到正确的地方。
